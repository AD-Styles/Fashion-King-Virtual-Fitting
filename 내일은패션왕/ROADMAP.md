# 로드맵

## Phase 1 — 체형 커스터마이징 (현재)

**목표**: 내 체형과 비슷한 마네킹을 만들고 빙글빙글 돌려볼 수 있다.

- [x] Next.js 14 + R3F + TypeScript 셋업
- [x] 임시 프리미티브 마네킹 (머리·몸통·팔·다리)
- [x] 체형 슬라이더 (키/몸무게/어깨/가슴/허리/엉덩이/다리)
- [x] OrbitControls + 자동 360° 회전 토글
- [ ] 실제 .glb 마네킹 모델 로딩 (`public/models/mannequin/`)
- [ ] Morph target 기반 체형 변형 (`shoulder_wide`, `waist_slim` 같은 블렌드셰이프)
- [ ] 체형 프리셋 ("표준 한국인 남성/여성", "슬림", "근육질" 등)
- [ ] 카메라 프리셋 (정면/측면/후면 버튼)

**완료 기준**: 사용자가 키 170, 어깨 +10% 같은 값을 넣으면 그에 맞게 마네킹이 즉시 반영되고 360° 부드럽게 돌아간다.

## Phase 2 — 옷 입히기

**목표**: 마네킹에 셔츠/바지/원피스 등을 갈아입히고 회전할 수 있다.

- [ ] 옷 카테고리 정의 (top, bottom, dress, shoes, accessory)
- [ ] 옷 .glb 모델 로딩 + 마네킹 본(bone)에 부착 (스켈레톤 매칭)
- [ ] 옷 선택 UI (썸네일 그리드, 카테고리별 필터)
- [ ] 옷 색상 변경 (머터리얼 컬러 조정)
- [ ] 체형 변경 시 옷이 자연스럽게 따라 늘어남 (스키닝)
- [ ] 룩 저장 (현재 체형 + 옷 조합을 로컬 스토리지에)
- [ ] 스크린샷 다운로드 버튼

**완료 기준**: 마네킹에 옷 3~5벌을 자유롭게 바꿔 입히고, 체형을 바꿔도 옷이 깨지지 않는다.

## Phase 3 — AI 통합 (옷 생성)

**목표**: 옷 사진 한 장을 업로드하면 AI가 3D 옷 메쉬로 변환해 마네킹에 입혀준다.

- [ ] AI 옷 생성 파이프라인 설계 (`docs/AI-INTEGRATION.md` 참고)
  - 옵션 A: 옷 이미지 → 텍스처만 추출 → 기성 옷 메쉬에 적용
  - 옵션 B: 옷 이미지 → 3D 메쉬 생성 (TripoSR, Wonder3D 등)
- [ ] API 라우트 (`src/app/api/generate-clothing/route.ts`)
- [ ] Replicate/Hugging Face/자체 GPU 중 선택
- [ ] 비용/지연 관리 (큐, 캐싱)

## Phase 4 — 개인화 (사용자 = 마네킹)

**목표**: 사용자 사진을 업로드하면 마네킹이 사용자 얼굴/체형을 닮게 된다.

- [ ] 사진 → 체형 추정 (MediaPipe BodyPose 또는 SMPL-X 기반)
- [ ] 사진 → 얼굴 텍스처 추출 후 마네킹 머리에 매핑
- [ ] 프라이버시 정책 (사진 로컬 처리 vs 서버 전송)
- [ ] "연예인 룩 따라하기" 모드 (연예인 사진 기반 룩 추천)

## 보너스 / 검토 중

- 모바일 앱 (React Native + react-three/fiber-native 또는 별도 Flutter 클라이언트)
- 공유 기능 (URL로 룩 공유)
- 친구와 비교 (둘 다 같은 옷 입혀보기)
- 쇼핑몰 연동 (옷 입혀본 후 구매 링크)
